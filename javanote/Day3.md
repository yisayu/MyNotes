###03.19_Java语言基础(选择结构switch语句的格式及其解释)(掌握)
* A:switch语句的格式
* 
		switch(表达式) { 表达式数据可以用可以提升为int的类型 byte short char int 1.7版本及之后String 也可以用
		      case 值1：
				语句体1;
				break;
			    case 值2：
				语句体2;
				break;
			    …
			    default：	
				语句体n+1;
				break;
	    }
 

* C:执行流程
	* 先计算表达式的值
	* 然后和case后面的匹配，如果有就执行对应的语句，否则执行default控制的语句

###03.21_Java语言基础(选择结构switch语句的注意事项)(掌握)
* A:案例演示
	* a:case后面只能是常量，不能是变量(记住)，而且多个case后面的值不能出现相同的 表达式可以是 byte char short int 引用数据类型可以用 String 和 枚举
	* b:default可以省略吗?
		* 可以省略，但是不建议，因为它的作用是对不正确的情况给出提示。
		* 特殊情况：
			* case就可以把值固定。
			* A,B,C,D
	* c:break可以省略吗?
		* 最后一个可以省略,其他最好不要省略
		* 会出现一个现象：case穿透。也可利用穿透节省代码量
		* 最终我们建议不要省略
	* d:default一定要在最后吗?
		* 不是，可以在任意位置。但是建议在最后。
	* e:switch语句的结束条件
		* a:遇到break就结束了
		* b:执行到switch的右大括号就结束了
